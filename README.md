# 📄 MyResume

Este é o repositório do meu currículo pessoal, desenvolvido para ser uma página web interativa e responsiva. O objetivo é consolidar minhas informações acadêmicas, habilidades técnicas e portfólio de projetos em um só lugar.

## 🚀 Tecnologias Utilizadas

O projeto foi construído utilizando tecnologias web fundamentais:

* **HTML5:** Estruturação de conteúdo.
* **CSS3:** Estilização, layout (Flexbox) e responsividade.
* **JavaScript:** Lógica para o sistema de abas interativas.

## 🛠️ O que você vai encontrar aqui?

* **Sobre Mim:** Um breve resumo da minha trajetória.
* **Educação:** Detalhes da minha formação na **Fatec Ribeirão Preto**.
* **Habilidades:** Stack técnica focada em **C# (.NET 9.0/10.0)**, SQL e Web.
* **Projetos:** Lista dos meus principais desenvolvimentos, incluindo sistemas de gestão e jogos.
